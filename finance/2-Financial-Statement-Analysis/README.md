# 2-Financial Statement Analysis

这个项目把“财务诸表分析入门”常见的核心知识点拆成了几个适合初学者练手的小项目，并提供了一个简单的 Python 程序，帮助你查看知识点、查看项目方案，以及进行基础财务分析。

## 讲义知识点拆解

下面这些内容通常是“财务诸表分析入门”的核心模块：

- 资产负债表：看企业资产、负债、所有者权益的结构
- 利润表：看收入、成本、利润的形成过程
- 现金流量：看企业现金流入流出与利润是否匹配
- 流动性分析：如流动比率、速动比率
- 偿债能力分析：如资产负债率、利息保障倍数
- 盈利能力分析：如净利率、ROA、ROE
- 营运能力分析：如总资产周转率、应收账款周转率
- 成长性分析：如营业收入增长率、净利润增长率
- 杜邦分析：把 ROE 拆成利润率、周转率、权益乘数
- 趋势分析与横向比较：比较企业不同年份或不同企业的表现

## 可做的小项目

### 1. 财务比率自动计算器

目标：
输入企业的关键财务数据，自动输出常见财务比率。

实现方案：

- 输入营业收入、净利润、总资产、流动资产、流动负债等基础数据
- 自动计算流动比率、资产负债率、净利率、ROA、ROE
- 适合作为最基础的分析工具

### 2. 企业趋势分析器

目标：
输入企业连续几年的收入和利润数据，判断企业增长趋势。

实现方案：

- 输入多个年份的数据
- 计算每年的增长率
- 输出“上升”“下降”“波动”这类简单判断
- 可以进一步扩展为图表版本

### 3. 杜邦分析小工具

目标：
把 ROE 拆解成几个组成部分，分析利润来源。

实现方案：

- 输入净利润、营业收入、总资产、股东权益
- 计算净利率、总资产周转率、权益乘数
- 输出 ROE 及其拆解结果
- 适合用来理解盈利能力背后的结构

### 4. 财务健康度评分器

目标：
根据几项关键指标，对企业做一个简单评分。

实现方案：

- 输入流动比率、资产负债率、净利率、ROE 等
- 设定简单阈值规则
- 输出“稳健”“一般”“风险偏高”等结论
- 适合作为规则型入门项目

## 程序功能

`financial_statement_project_planner.py` 提供以下功能：

- 查看知识点拆解
- 查看每个项目的简介、输入、输出和实现步骤
- 计算常见财务比率
- 计算趋势增长率
- 进行杜邦分析
- 根据几个指标生成简单的财务健康判断

另外，这个目录里还提供了一个 Excel 模板生成脚本 `generate_financial_excel.py`，会生成一个双语 Excel：

- `financial_analysis_template_bilingual.xlsx`

这个 Excel 模板包含 3 个 sheet：

- `Data`（隐藏）
- `English`
- `Japanese`

其中：

- `Data` 是统一数据源
- `English` 和 `Japanese` 都联动到 `Data`
- 两个展示 sheet 的数字和结果会自动同步

补充说明：

- 目前这是“不用宏”的稳定联动方案
- 如果你需要“在 English 或 Japanese 任意一边都能直接输入，并自动反写到另一边”，那就需要做成带 VBA 的 `.xlsm`

模板已经嵌入了资产负债表与核心比率公式，包括：

- 总资产
- 总负债
- ROE
- ROA
- 净利率
- 流动比率
- 资产负债率
- 负债权益比
- 总资产周转率
- 权益乘数
- 杜邦分解校验

## 运行方式

```bash
python financial_statement_project_planner.py
```

如果你要生成 Excel 模板，也可以运行：

```bash
python generate_financial_excel.py
```
