# 1-Time Value

このプロジェクトは、お金の時間価値に関する講義内容を計算できる Python プログラムです。

含まれているモジュール:

- 時間軸とキャッシュフロー表
- 将来価値 (`FV`)
- 現在価値 (`PV`)
- 正味現在価値 (`NPV`)
- 永久債
- 成長型永久債
- 年金現価
- 年金終価
- 成長型年金現価
- 借入金の毎期返済額
- 借入残高
- 内部収益率 (`IRR`)

## 実行方法

```bash
python time_value_calculator.py
```

起動後に次のどちらかを選べます。

- 対話モード: 変数を自分で入力して計算
- デモモード: あらかじめ用意した例で計算結果を確認

## 注意

- 利率は小数で入力してください。たとえば `5%` は `0.05` と入力します。
- キャッシュフローは `t0` から順に、カンマ区切りで入力してください。例: `-1000,300,400,500`
